# Stellar Jarvis Sidecar

This folder stores persistent memory for the **Stellar Jarvis** Expert agent.

## Purpose
Requires immediate context loading of desk state, trade logic, and financial forecasts to function as an Active Observer.

## Files
- memories.md: Desk Pulse state and user interactions
- instructions.md: Operational protocols and trade logic
- knowledge/2026-forecasts.json: Financial targets

## Runtime Access
After BMAD installation, this folder will be accessible at:
`{project-root}/_bmad/_memory/stellar-jarvis-sidecar/{filename}.md`
