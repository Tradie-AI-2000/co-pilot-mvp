# Stellar Recruitment Co-Pilot - Walkthrough

I have successfully built the **Stellar Recruitment Co-Pilot**, a premium web application designed to assist with your labour hire and recruitment workflows.

## Features Implemented

### 1. Intelligent Dashboard
- **Overview**: A central hub showing real-time metrics.
- **Widgets**:
  - **Active Jobs**: Urgent and active roles with status indicators.
  - **Market Pulse**: Quick view of regional demand.
  - **Recent Placements**: Track your latest wins and fees.

### 2. CRM (Client Relationship Management)
- **Client List**: View key accounts like Fletcher Construction and Downer.
- **Status Tracking**: Monitor "Key Account", "Active", and "At Risk" statuses.
- **Activity**: See active jobs and last contact times.

### 3. Candidate Pool
- **Smart Search**: Placeholder for searching by skill or location.
- **Profiles**: Rich cards showing candidate roles, ratings, and availability.
- **Status**: Quickly identify who is "Available" vs "On Job".

### 4. Market Intelligence
- **Heat Map**: Visual representation of labour demand across Auckland (Placeholder).
- **Regional Insights**: Detailed breakdown of demand trends and top roles by region.

## Design System
- **Premium Aesthetic**: Deep midnight blue background with electric blue and gold accents.
- **Glassmorphism**: Translucent panels for a modern, high-end feel.
- **Responsive**: Works on different screen sizes.

## How to Run

1. Open the terminal.
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
